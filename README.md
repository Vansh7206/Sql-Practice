# SQL Practice

Practicing SQL concepts topic-wise using realistic datasets. Each folder contains questions with solutions of increasing difficulty.

## Topics Covered
- **Basics** — SELECT, WHERE, ORDER BY, LIMIT, LIKE, BETWEEN, DISTINCT
- **Aggregations** — SUM, AVG, COUNT, GROUP BY, HAVING, EXTRACT
- **Joins** — INNER, LEFT, RIGHT, FULL JOIN, multi-table JOINs
- **Subqueries** — Nested queries, IN, NOT IN, correlated subqueries

## Structure
Each folder contains:
- Practice questions as SQL comments
- Solutions written and tested in pgAdmin (PostgreSQL)

## Status
Work in progress — new topics added daily as I practice.
